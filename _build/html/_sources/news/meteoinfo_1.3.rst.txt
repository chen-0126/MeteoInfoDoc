******************************************
MeteoInfo 1.3 was released (2016-1-1)
******************************************

MeteoInfo 1.3 was released. A color map dialog was added for users to choose color map easily. 
Plugin applicaitons were supported, and a IMEP (Integrated Model Evaluation Platform) application
was added. Also the verification functions were added in toolbox module.
