**************************
Cluster Statistics
**************************

Click ‘Cluster Statistics’ menu item, ‘Cluster Statistics’ dialog will be opened. In which, field, cluster number, 
pollution criterion and missing value should be set.

.. image:: ../../_static/trajstat/cluster_stat.png

Then press ‘OK’ button, the statistical result will be showed with trajectory number, mean value, standard deviation, 
polluted trajectory number, polluted mean value and polluted standard deviation of each cluster.

.. image:: ../../_static/trajstat/cluster_stat_result.png
