**************************
Select Trajectory
**************************

The trajectories could be selected according to the attribute using 'Selction -> Select by Attribute' menu. In this 
example, we want to select the trajectories with PM10 concentration larger than 150 μg m-3 which is Class Ⅱ Chinese 
standard.
