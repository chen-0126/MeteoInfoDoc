*******************
xlim
*******************

.. currentmodule:: mipylib.plotlib.miplot

.. function:: xlim(xmin, xmax)

    Set the *x* limits of the current axes.
    
    :param xmin: (*float*) Minimum limit of the x axis.
    :param xmax: (*float*) Maximum limit of the x axis.
    
    **Example:**
    
    ::

        >>> xlim(2, 12)
