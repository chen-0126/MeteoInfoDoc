MeteoInfo 1.4 was released (2017-1-15)Â¶

MeteoInfo 1.4 was released. Code auto-completion function was added in MeteoInfoLab Editor.

mipylib package was re-arranged to several sub-packages: numeric, dataset, plotlib, geolib and meteolib.

Several linear algebra functions were added: cholesky, lu, qr, svd, eig, solve powered by Apecha Common Math library.
